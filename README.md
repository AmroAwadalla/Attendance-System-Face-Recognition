# Attendance-System-Face-Recognition
AI-powered attendance management system using Python, OpenCV, and MySQL for real-time facial recognition and automated attendance tracking.

## Overview

This project is an AI-powered attendance management system developed as my Bachelor's graduation project in Computer Engineering.

The application uses Computer Vision techniques to recognize registered students and automatically mark attendance using a webcam.

---

## Features

- Student Registration
- Face Image Capture
- Face Recognition
- Automatic Attendance Recording
- Attendance Viewer
- Attendance Report Generation
- MySQL Database Integration
- Modern GUI using CustomTkinter

---

## Technologies

- Python
- OpenCV
- CustomTkinter
- NumPy
- Pandas
- Pillow
- MySQL

---

## System Workflow

1. Register Student
2. Capture Face Images
3. Train Recognition Model
4. Detect Face
5. Recognize Student
6. Save Attendance
7. Export Attendance Report

---

## Installation

Clone repository

```bash
git clone https://github.com/AmroAwadalla/Attendance-System-Face-Recognition.git
```

Install packages

```bash
pip install -r requirements.txt
```

Run

```bash
python main.py
```
